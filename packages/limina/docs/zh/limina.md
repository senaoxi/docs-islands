# 什么是 Limina

Limina 是面向 TypeScript 单体仓库的架构治理工具。

它的接入路径很明确：先接入增量构建，再逐步打开架构治理。团队可以先基于现有 TypeScript 配置和源码依赖关系，生成可复用的类型构建配置，获得稳定的构建入口。随后再把同一组工程事实延伸到依赖图治理、源码边界检查、检查覆盖验证和发布前检查。

对于小型项目来说，直接运行 TypeScript 类型检查通常已经足够。但在大型单体仓库中，工程关系会更复杂：

- 多个包之间需要通过明确的依赖声明相互访问。
- TypeScript 项目引用需要和真实源码导入保持一致。
- TypeScript、Vue、Svelte、文档、脚本和测试代码可能需要不同的检查方式。
- 发布到 npm 的包还需要正确的入口、类型声明、依赖声明和发布内容。

这些信息通常分散在 `tsconfig`、`package.json`、源码导入、检查器配置和发布产物中。随着仓库规模增长，它们很容易逐渐偏离真实状态。

Limina 的作用不是替代这些工具，而是把它们依赖的工程关系变成可生成、可校验、可运行的检查。

## 先从增量构建接入

第一次接入 Limina 时，推荐先从类型构建开始，而不是一开始就打开所有治理规则。

Limina 会读取现有 TypeScript 配置和源码依赖关系，生成可复用的类型构建配置，并安排合理的构建顺序。团队不需要大幅调整工程结构，就可以先获得一个可运行、可增量、适合放进本地开发和持续集成流程的构建入口。

这条路径的价值在于：

- 先获得明确的工程收益，而不是先被完整规则体系拦住。
- 构建关系由源码配置和真实导入推导，减少长期手动维护成本。
- 等类型构建稳定后，再逐步启用依赖图治理、源码边界检查和检查覆盖验证。

对于大型 TypeScript 单体仓库，这通常是成本较低、收益也更容易验证的接入方式。

## 再逐步打开架构治理

当增量构建路径稳定后，团队可以继续启用更多检查，把仓库中的工程约束固定下来。

Limina 的核心治理能力包括以下几类。

**治理依赖图。** 检查真实源码导入、TypeScript 项目引用和工作区依赖声明是否一致，发现缺失引用、额外引用、非法访问和未声明依赖。

**保护源码边界。** 检查跨包相对导入、未授权导入、漏写依赖和源码归属问题，避免模块绕过预期入口直接访问内部实现。

**验证检查覆盖。** 找出没有被检查覆盖、被重复覆盖，或检查范围与源码范围不一致的文件，减少质量盲区。

**编排检查流程。** 构建、依赖图检查、源码边界检查和检查覆盖验证都可以作为独立任务运行，也可以组合成本地开发、持续集成或发布前流程。能够并发执行的任务会尽可能并发完成，减少等待时间。

**补充发布前检查。** 在发布前验证包元数据、类型入口、构建产物和打包内容，提前发现可能影响使用方的问题。

这些能力不需要一次性全部启用。团队可以按仓库阶段逐步接入：先让构建跑稳，再把关键边界放进变更检查，最后把发布前检查接入发布流程。

## 适合哪些项目

Limina 更适合已经进入工程复杂度阶段的 TypeScript 单体仓库，尤其是以下情况：

- 使用 pnpm 工作区管理多个包。
- 已经使用 TypeScript 项目引用，或准备采用增量类型构建。
- 希望减少手写和同步项目引用的成本。
- 需要约束生产代码、工具代码、测试代码、浏览器代码和 Node 代码之间的访问边界。
- 仓库中同时存在 TypeScript、Vue、Svelte 或其他需要专属检查器的代码。
- 会发布 npm 包，并希望在发布前验证实际产物。

如果项目规模较小，包边界简单，也没有发布产物需要检查，直接使用 TypeScript 和现有检查工具可能已经足够。

## Limina 不是什么

Limina 不是打包器、测试框架或发布工具，也不会替代 TypeScript 或框架专属检查器。

它运行在这些工具旁边，负责验证它们依赖的单体仓库结构是否仍然可靠：源码关系是否合理，类型构建图是否一致，检查范围是否完整，发布产物是否符合预期。

换句话说，Limina 关注的不是业务代码怎么写，而是大型仓库中的工程约束是否仍然清晰、可审查、可持续维护。

## 接入后的工作方式

接入 Limina 后，团队可以把不同检查放到不同阶段。

本地开发时，使用增量构建快速验证类型构建关系是否仍然可用。

持续集成阶段，检查依赖关系、源码边界和检查覆盖，避免架构漂移随着代码变更进入主干。

发布前，验证包元数据、类型入口、构建产物和打包内容，减少发布后才暴露的问题。

最终效果是：当一次变更破坏了工程关系时，失败信息会更明确地指向问题来源。团队可以判断这次应该补项目引用、声明依赖、调整源码边界、补充检查覆盖，还是修复发布产物。
